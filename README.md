## Avoid the Cactuses
Orange cactuses come from the right. Jump over them by making a sharp loud sound at the right moment. Don't jump too early or too late — timing is everything fr.

Every 100 points the speed increases. It gets genuinely unhinged after score 200 💀

---

## Scoring
- Pass a cactus without hitting = +points + anime wow sound 🎵
- Hit a cactus = game over + cat laugh sound 😭
- Tap anywhere on game over screen to restart and pick mode again

---

## Tips
- Short sharp sounds work best for jumping — think clap, snap, or "HEY!"
- In Silent Mode even a whisper triggers a jump so stay quiet between jumps
- In Rage Mode you genuinely need to scream — your neighbors will know 💀
- Watch the **VOICE METER** bar at the bottom to see how loud you're being in real time

## What's New in v5.0

This update is the premium-feel pass. I wanted the game to stop feeling like a small prototype and start feeling like a real, polished arcade game with better response, better visuals, and more reasons to keep playing.

- Added ultra-realistic flying enemies that start appearing after score 300.
- Added sky raid wave logic so late-game runs feel more intense and unpredictable.
- Character and cactus shadows now appear only when the sun is out, so the lighting finally makes visual sense.
- Reduced jump latency across Silent, Normal, and Rage modes so the character reacts immediately when a jump sound is detected.
- Added roof bounce behavior so screaming too loudly no longer sends the character out of the phone screen.
- Added a full in-game settings menu with mode selection, sensitivity tuning, HUD opacity, graphics, lighting, audio, gameplay, and stats controls.
- Added device-friendly voice calibration so different microphones can be tuned more fairly.
- Upgraded visuals with score-driven day/night lighting, premium desert scenery, clouds, mountains, dunes, particles, bloom, screen effects, and cleaner UI.
- Added flying enemies, improved obstacle rendering, better character animation, responsive haptics, sharing support, and stronger performance settings.
- Cleaned up duplicated renderer and particle code so the project is easier to maintain.

## 🛠 Tech Stack
=======
1. Clone the repository:
>>>>>>> f193447 (v5.0: add flying enemies, improve voice latency, implement ceiling bounce, add settings menu)

```bash
git clone https://github.com/ShounakPatra/vocex-run.git
```

2. Open the project in Android Studio.
3. Let Gradle sync the project.
4. Connect an Android device or start an emulator.
5. Build and run the `app` configuration.

On Windows, you can also verify the Java build from PowerShell:

```powershell
.\gradlew.bat compileDebugJavaWithJavac --console=plain
```

## Tech Stack

- Java
- Android SurfaceView
- AudioRecord microphone input
- SoundPool sound effects
- Custom game loop
- AndroidX FileProvider sharing

## Notes

- Microphone permission is required.
- The game is best played in a quiet room, unless you are intentionally choosing Rage mode.
- Use the sensitivity slider if your phone microphone is too quiet or too aggressive.

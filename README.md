## 🎮 Overview

Vocex Run is a real-time voice-controlled endless runner where your microphone replaces traditional controls.

- 🤫 Silence slows movement
- 🗣️ Speaking increases speed
- 💥 Sharp sounds trigger jumps

> There are no buttons. No touch controls. Only your voice.

---

## ⚙️ Core Gameplay

| Voice Input | Action |
|---|---|
| Stay silent | Slow movement |
| Speak normally | Faster movement |
| Sharp sound | Jump |
| High intensity | Maximum speed |

Precision matters more than volume. Timing is everything.

---

## 🧠 Game Modes

| Mode | Best For |
|---|---|
| 🔇 Silent Mode | Quiet environments |
| 🎙️ Normal Mode | Everyday use |
| 😤 Rage Mode | Noisy surroundings |

Each mode adjusts voice thresholds for consistent gameplay across environments.

## 🛠️ How to Run

```bash
git clone https://github.com/ShounakPatra/vocex-run.git
```

1. Open in **Android Studio**
2. Connect your Android device via USB
3. Grant microphone permission when prompted
4. Hit **Run ▶**

---

## ⚠️ Requirements & Notes

- Android device with a working microphone
- Microphone permission must be granted at runtime
- Performance may vary depending on device microphone sensitivity

## 🎮 Gameplay

Control your character using your **voice intensity**:

- 🤫 **Whisper** → slow movement  
- 🗣️ **Normal** → run  
- 😱 **Shout** → jump  
- 💀 **Rage** → chaos mode (screen effects + madness)

🔥 The louder you go, the more control you get… or lose.

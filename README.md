# 🎤 Voice Rage Game

A chaotic voice-controlled Android game where your **loudness controls movement** 😈
## 📲 Download Game

👉 [Download APK](https://github.com/ShounakPatra/voice-rage-game/releases)

## 🎮 Gameplay

* 🤫 Whisper → slow movement
* 🗣️ Normal → run
* 😱 Shout → jump
* 💀 Rage → chaos mode (screen shake + madness)

## 🔥 Features

* Voice-controlled physics
* Dynamic difficulty scaling
* Sound effects (MLG, bruh, etc 💀)
* Screen shake + rage visuals
* Obstacle system with fair spawning

## 🛠 Tech Stack

* Java (Android)
* SoundPool API
* Microphone input (AudioRecord)
* Custom game loop

## 🚀 How to Run

1. Clone the repo
2. Open in Android Studio
3. Connect your phone
4. Run ▶

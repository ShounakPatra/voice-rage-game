## 🎮 Overview

Vocex Run is a real-time voice-controlled endless runner where:

* Silence slows you down
* Speech increases speed
* Sharp sounds trigger jumps

There are no buttons. No touch controls.
Only your voice.

---

## ⚙️ Core Gameplay

* **Stay silent** → Slow movement
* **Speak normally** → Increased speed
* **Sharp sound** → Jump
* **High intensity voice** → Maximum speed state

Precision matters more than volume. Timing is everything.

---

## 🧠 Game Modes

* **Silent Mode** — Designed for quiet environments
* **Normal Mode** — Balanced for everyday surroundings
* **Rage Mode** — Tuned for noisy environments

Each mode adjusts detection thresholds for consistent gameplay.
